Appointment Types
About Types

Appointment types let you categorize and color-code appointments. Set a follow-up interval to auto-suggest the next visit.

💡 Tip
For Oral Prophylaxis, set 6 months. For Orthodontic Adjustment, set 1 month. The app will suggest the follow-up date automatically.
📱 WhatsApp Reminders

Send appointment reminders via WhatsApp — free, no setup needed. Opens WhatsApp with a pre-written message ready to send.

✅ Already active
WhatsApp reminders work on all appointments. Open any appointment and click "Send WhatsApp Reminder".
📨 Semaphore SMS

Send real SMS to patients via Semaphore (Philippine gateway). Works on Globe and Smart numbers.

⚙️ Setup required
Sign up at semaphore.co → go to Account → API Keys → copy your key and paste below.
🗄️ Archive Old Records

Appointments older than 1 year are moved to a hidden Archive tab in your Google Sheet automatically on every sync — keeping the main sheet fast and within quota, while preserving all records permanently.

Auto-archiving also runs silently on every sync. Archived records are never deleted — they stay in the Archive sheet tab forever.
